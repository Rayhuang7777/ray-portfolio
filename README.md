## Local Development

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/ray-portfolio.git
```

Open:

```bash
index.html
```

or run locally:

```bash
python -m http.server
```

Then visit:

```txt
http://localhost:8000
```
